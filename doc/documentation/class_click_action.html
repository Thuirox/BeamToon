Detailed Description

Abstract class used to implement class that will hold a function to be executed. Those classes are meant to be linked to a Drawable object. And therefore represent the action that is triggered when the visual object is clicked on. Furthermore due to the fact that multiple Drawable can be stacked, each ClickAction determine if those of the others object has to be triggered to or not.
